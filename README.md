Implementação de plataforma de desenvolvimento de jogos para auxílio no ensino de programação em sistemas embarcados.
Trabalho de conclusão apresentado ao Curso de Ciência da Computação da Universidade Federal de Viçosa como requisito para obtenção do título de bacharel em Ciência da Computação.
    Este trabalho propõe o desenvolvimento de uma interface de programação voltada à criação de jogos simples em plataformas microcontroladoras, com o objetivo de apoiar atividades práticas em cursos introdutórios de programação, além de servir como elemento de exibição em eventos, estimulando e convidando o indivíduo a iniciar o processo de aprendizado. A proposta parte da observação de que, embora atividades de cursos de programação tradicionais cumprirem seu papel de guiar o aprendizado, desde que bem estruturadas, vê-se a possibilidade de potencializar a efetividade desse processo, o enriquecendo com elementos de feedback visual claros, problemas com aplicação de lógica clara e direta e um ambiente lúdico.
